# 📌 Chill Money (Учёт расходов)

**Мобильное приложение для учёта расходов и доходов.**  
Простое и удобное решение, которое помогает хранить, анализировать и управлять финансовой информацией прямо с телефона.

---

## 🚀 Функционал  

✨ **Основные возможности:**  
- 📊 Добавление и управление бюджетами для контроля расходов и доходов  
- ➕ Добавление транзакций (трата или получение) с указанием суммы и названия  
- ⚙️ Настройки для персонализации приложения  
- 📈 Просмотр статистики доходов и расходов за месяц  
- 🕒 История транзакций с возможностью анализа прошлых операций  

📱 **Платформы:**  
- Android  
- iOS
